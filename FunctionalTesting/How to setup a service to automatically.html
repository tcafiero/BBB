How to setup a service to automatically run a python script when the BeagleBone Black (BBB) reboots. These instructions allow you to setup a python script to automatically start when the BeagleBone (running Angstrom linux with systemd) restarts.
Raw
bbb_boot_service_instructions.md

Creating a service to startup at BeagleBone Black boot time:

  • Use shell script /usr/src/BBB/FunctionalTesting/StartSocatService.sh:

    #!/bin/bash

# this could be any runnable code or shell script, really
socat TCP-LISTEN:7777,fork /dev/ttyUSB0,raw,echo=0,b115200

Note that the first line is critical.

  • Copy /usr/src/BBB/FunctionalTesting/IoThingsWareSocat.service in /lib/systemd/ IoThingsWareSocat.service is such as:

    [Unit]
Description=IoThingsWare Socat Service
After=syslog.target network.target

[Service]
Type=simple
ExecStart=/usr/src/BBB/FunctionalTesting/StartSocatService.sh

[Install]
WantedBy=multi-user.target

  • Create a symbolic link between your script and a special location under /etc:

    ln /lib/systemd/IoThingsWareSocat.service /etc/systemd/system/IoThingsWareSocat.service

  • Make systemd aware of your new service

    systemctl daemon-reload
systemctl enable IoThingsWareSocat.service
systemctl start IoThingsWareSocat.service

  • Reboot the BeagleBone Black to see your script in action

  • If you wish to control the service at runtime, you can use systemctl commands: 

    systemctl status IoThingsWareSocat.service
systemctl stop IoThingsWareSocat.service
systemctl start IoThingsWareSocat.service
systemctl disable IoThingsWareSocat.service
